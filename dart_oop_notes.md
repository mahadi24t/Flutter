# 📘 Dart OOP Quick Notes (Day 1)

## 🏗️ 1. Class (নকশা)
যেকোনো জিনিস বানানোর আগে তার একটা নকশা লাগে। কোডিংয়ের ভাষায় এই নকশাই হলো **Class**।
* **উদাহরণ:** `class Robot { ... }`

## 🤖 2. Object (আসল জিনিস)
নকশা দেখে তৈরি করা সত্যিকারের জিনিসটাই হলো **Object**। ফ্লাটারে আমরা যত Text, Button দেখি, সবই এক একটা Object।
* **উদাহরণ:** `Robot myRobot = Robot();`

## 📝 3. Variables (বৈশিষ্ট্য)
কোনো কিছুর তথ্য বা ডেটা ধরে রাখা। যেমন রোবটের নাম, রং বা বয়স।
* **উদাহরণ:** `String name = "AI Detective";`

## ⚙️ 4. Methods (কাজ)
কোনো অবজেক্ট কী কী কাজ করতে পারে (হাঁটা, কথা বলা, ডেটা চেক করা)।
* **উদাহরণ:** `void speak() { ... }`

## 🧬 5. Inheritance (`extends`)
পুরোটা নিজে না বানিয়ে অন্যের তৈরি করা কোনো নকশার সুবিধা নেওয়া বা ধার করা। 
* **উদাহরণ:** `class MyPage extends StatelessWidget { ... }`

## 🛑 6. `@override` (নিজের নিয়ম চাপানো)
যার কাছ থেকে সুবিধা নিচ্ছি, তার কোনো পুরনো নিয়ম বাতিল করে নিজের মতো করে নতুন নিয়ম লিখে দেওয়া।
